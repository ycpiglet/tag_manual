# 자동문 MtoM IoT 매뉴얼

## 개요

자동문 제어 모듈은 로봇의 출입 동선을 지원하기 위한 MtoM IoT 장치다.

## 기동 절차

1. TOM 전원을 ON 한다.  
2. `door_robot.py` 실행 상태를 확인한다.  
3. 자동문 센서 상태를 확인한다.  
4. Rubberneck 연결 상태를 확인한다.

## 점검 체크리스트

- TOM 상태
- 자동문 센서 상태
- 제어 모듈 상태
- Rubberneck 연결 상태
- 수동 오버라이드 가능 여부

## 수동 제어

- Rubberneck에서 자동문을 선택한다.
- 개방/폐쇄 명령을 전송한다.
- 로봇 통과 후 문 상태를 확인한다.
