## 2. 전원 켜기/끄기

### 켜기 순서

> 총 소요 시간: 약 2~3분 (부팅 포함)

1. 로봇 **후면 덮개 열쇠**로 배터리 덮개 열기
2. **트리거 스위치** → `I` (켜짐) 확인 (기본값, 건드릴 필요 없음)
3. **하드웨어 전원 스위치** → `ON` 위치로 올리기
4. 배터리 덮개 닫기
5. **제어 버튼 덮개** 열고 **소프트웨어 Power 버튼 3초 이상** 누르기
6. 부팅 완료 후 **Start 버튼 LED가 녹색으로 깜빡임** → **Start 버튼** 누르기
7. 로봇 작동 시작

> ⚠️ Start 버튼을 누르기 전까지 로봇은 명령을 받지 않음

### 끄기 순서 (일반)

1. **소프트웨어 Power 버튼 3초 이상** 누르기
2. LED가 완전히 꺼질 때까지 약 **1분 대기**

### 끄기 순서 (장기 미사용 또는 점검 시)

1. 소프트웨어 전원 끄기 (위 일반 절차)
2. 배터리 덮개 열기
3. **하드웨어 스위치** → `OFF`
4. **트리거 스위치** → `O` (꺼짐)
5. 배터리 덮개 닫기

> 장기 보관 시 트리거까지 끄지 않으면 배터리 자가 방전됨

## 3. 비상 정지

### 활성화
- 로봇 **좌측 또는 우측 비상 정지 버튼** 누르기
- 즉시 모터 전원 차단, 로봇 정지

### 해제 및 재기동
1. 비상 정지 버튼을 **시계 방향으로 돌려** 잠금 해제
2. **Start 버튼 LED가 녹색 깜빡임** 확인
3. **Start 버튼** 누르기 → 정상 복귀

> ⚠️ 비상 정지 해제 전에 주변 안전 확인 필수

## 4. 브레이크 스위치

| 스위치 상태 | 의미 | 로봇 동작 |
|-------------|------|-----------|
| `ON` (활성화) | 브레이크 작동 | 자율/수동 주행 모두 가능 |
| `OFF` (비활성화) | 모터 전원 차단 | 주행 불가, **손으로 밀어서 이동만 가능** |

**수동 이동이 필요한 경우:**
1. 로봇이 **정지 상태**인지 확인
2. 브레이크 스위치 → `OFF`
3. 손으로 천천히 밀기 (급하게 밀면 감속 충격 발생)
4. 이동 완료 후 반드시 `ON`으로 복구

> 브레이크가 OFF인데 로봇이 안 움직이면: 브레이크 스위치를 ON으로 확인 후 다시 시도

## 5. 조이스틱 조작 (Logitech)

### 준비

1. **Joystick 스위치** → `Manual` 모드로 전환 (열쇠 사용)
2. **Start 버튼** 눌러 로봇 활성화
3. 조이스틱 전면의 **작동 스위치** → `X` 방향 확인 (`D`면 오작동 경고)

> ⚠️ 수동 주행 후 반드시 Joystick 스위치 열쇠를 제거하고 안전한 곳에 보관

### 버튼 배치

```
[LB] ← 데드맨 스위치 (항상 누른 상태로 사용)
[모드 버튼] ← D-pad / 제어 스틱 전환
[D-pad] ← 키패드 방향 조작
[좌 스틱] ← 아날로그 조작
[RT] ← 터보 모드 (최대 속도 증가)
[Logitech 버튼] ← SLAM/로컬라이제이션 모드 전환
[우 스틱] ← 기능 없음
```

### D-pad(키패드)로 조작

1. **모드 버튼** 눌러 **녹색 LED 켜기**
2. **LB 누른 상태**에서 **D-pad** 방향 입력

### 아날로그 스틱으로 조작

1. **모드 버튼** 눌러 **녹색 LED 끄기**
2. **LB 누른 상태**에서 **좌 스틱** 조작

### 매핑(지도 작성) 시 권장 절차

1. Joystick 스위치 → `Manual`
2. Start 버튼으로 로봇 활성화
3. Logitech 버튼으로 **SLAM(매핑) 모드** 선택
4. **로봇 뒤에 서서** 따라가며 공간 전체를 주행
5. 외곽 경로 먼저 → 내부 구역 순서로 매핑
6. 내부 구역이 **시작 지점(원점)을 통과**하도록 경로 설계
7. 매핑 완료 후 Logitech 버튼으로 **로컬라이제이션 모드** 전환

## 6. LED 상태 표시

로봇 둘레의 LED 밴드(1~8번 위치)가 상태를 표시.

<details>
<summary>주요 LED 상태 목록 (펼치기)</summary>

### 정상 상태

| LED 표시 | 의미 |
|----------|------|
| 하늘색 점등 (Stay On) | 자율 주행 중 |
| 하늘색 점등 (Stay On) | 수동 주행(Manual Drive) 중 |
| 하늘색 0.5s 깜빡임 | 전진/후진/좌회전/우회전 중 |
| 하늘색 0.3s 고속 깜빡임 | 고속 주행 중 |
| 하늘색 점등 | SLAM(매핑) 진행 중 |
| 녹색 6s 그라디언트 | 충전 중 |
| 녹색 점등 (Stay On) | 완충 완료 |
| 흰색 0.5s 깜빡임 | 부팅 중 |
| 흰색 점등 | 일시 정지(Paused) |
| 흰색 0.3s 깜빡임 | 사람 입력 대기 중 |
| 흰색 1s 깜빡임 | 시스템 트리거 대기 중 |

### 오류 상태

| LED 표시 | 의미 | 조치 |
|----------|------|------|
| 빨간색 점등 (Stay On) | 비상 정지 활성화 | 버튼 해제 후 Start |
| 빨간색 Cross Blink | 긴급 레벨 오류 | 재부팅 또는 지원팀 연락 |
| 빨간색 0.5s Cross Blink | FuSa(기능 안전) 오류 | Lidar/인코더/GPIO/Cliff 관련 |
| 빨간색 Burst | Stuck (장애물에 막힘) | 장애물 제거 후 Start |
| 빨간색 0.5s Blink | 도킹 실패 또는 BMS 오류 | 충전 위치 재확인 |
| 주황색 Gradient | 저배터리 | 즉시 충전 필요 |
| 노란색 Cross Burst | 로컬라이제이션 미완료 | 충전 스테이션 복귀 후 재시도 |

</details>

**자주 보는 상태 요약:**

| 색상 | 패턴 | 상황 |
|------|------|------|
| 🟢 녹색 | Gradient | 충전 중 |
| 🟢 녹색 | Stay On | 완충 |
| 🔵 하늘색 | Stay On | 정상 주행 |
| ⚪ 흰색 | 0.5s Blink | 부팅 중 |
| 🔴 빨간색 | Stay On | 비상 정지 |
| 🔴 빨간색 | Cross Blink | 시스템 오류 |
| 🟠 주황색 | Gradient | 배터리 부족 |

## 7. 충전

### 7.1 자동 충전 (정상 운영 시)

> 충전 스테이션 LED: **빨간색** = 대기, **주황색** = 충전 중, **녹색** = 완충

FMS(Balcony) 또는 Rubberneck을 통해 자동 도킹 명령 전송 → 로봇이 자동으로 충전 스테이션에 도킹.

**충전 스테이션 조건 확인:**
- 충전 모드 스위치 → `O` (Auto Mode)
- 전원 케이블 연결 확인
- 충전 단자와 로봇 단자 높이 일치 확인
- 충전 스테이션 주변 1800 × 1600 mm 공간 확보

### 7.2 수동 충전 (비상 시)

<details>
<summary>수동 충전 절차 (펼치기)</summary>

1. 로봇과 충전 스테이션 **전원 모두 끄기**
   - 로봇: 하드웨어 스위치 → `OFF`
   - 충전 스테이션: 전원 케이블 분리
2. 충전 스테이션 충전 모드 스위치 → `I` (Manual Mode)
3. 케이블 브라켓에서 **수동 충전 케이블** 분리
4. 수동 충전 케이블을 로봇 충전 커넥터에 연결  
   (**빨간색 점 표시 일치** 확인 후 연결)
5. 로봇과 충전 스테이션 전원 켜기
6. 충전 스테이션 LED → 주황색으로 변경 확인
7. 완전 충전 후 전원 끄기 → 케이블 분리 → 충전 모드 스위치 → `O` (Auto)로 복구

</details>

### 7.3 배터리 수동 충전기 사용 (충전 스테이션 사용 불가 시)

<details>
<summary>배터리 수동 충전기 절차 (펼치기)</summary>

**전압 확인**: GoCart180은 **29 V** 충전기 사용

1. 배터리 덮개 열기
2. 하드웨어 스위치 → `OFF`
3. 트리거 스위치 → `O`
4. 배터리 패널에서 **로봇 전원 케이블 커넥터 분리**
5. 배터리 수동 충전기 케이블 → **배터리 소켓** 연결
6. 충전기 전원 플러그 → 220V 콘센트 연결
7. 전원 어댑터 LED: **녹색 → 빨간색**으로 바뀌면 충전 시작  
   *(녹색 유지 시 배터리 불량, 교체 필요)*
8. 완전 방전 상태면 최소 **5시간 이상** 충전
9. 충전 완료 후 충전기 분리

</details>

## 8. 배터리 완전 방전 시 복구

> 증상: 배터리 잔량 0%, 소프트웨어 전원 버튼 무반응

1. 배터리 덮개 열기
2. 하드웨어 스위치 → `OFF`
3. 트리거 스위치 리셋:  
   `I(켜짐)` → `O(꺼짐)` → `I(켜짐)` 순서로 토글
4. 하드웨어 스위치 → `ON`
5. 배터리 덮개 닫기
6. **수동 충전 방식** ([7.2 수동 충전](#72-수동-충전-비상-시))으로 초기 충전 시작
7. **최소 30분 충전**, 배터리 전압 **24V 이상** 확인
8. 충전 완료 후 자동 충전 방식으로 전환

## 10. 안전 운영 기준

<details>
<summary>핵심 제한 사항 (펼치기)</summary>

### 하드웨어 한계
- 최대 적재량: **평지 180 kg / 경사·문턱·카펫 120 kg** (적재함 포함)
- 최대 주행 경사: **4.76°**
- 통과 가능 문턱: **10 mm 이하**
- 계단 추락 감지 최소 단차: **전면 150 mm / 후면 50 mm**  
  → 그 이하는 추락 감지 불가

### 2D 라이다 감지 한계
- 장애물 최소 기준: 지름 **70 mm**, 높이 **100 mm** 이상
- 투명한 유리벽/문은 미감지 가능

### 권장 환경
- 온도: **5°C ~ 35°C**
- 습도: **최대 80%**
- 조도: **300 ~ 1,000 Lux** (충전 스테이션 마커 인식용)
- 운영 구역: **실내 전용**

### 금지 사항
- 습하거나 젖은 바닥 주행
- 4.76° 이상 경사면 주행
- 에스컬레이터 근처 운영
- 150 mm 미만 단차 높이의 계단 근처 운영
- USB 포트에 5W 초과 외부 장치 연결

</details>

## 11. 트러블슈팅

<details>
<summary>문제별 해결 방법 (펼치기)</summary>

### 로봇이 Start 버튼 누른 후에도 움직이지 않음

1. **브레이크 스위치** → `ON` 확인
2. **Joystick 스위치** → `Auto` 확인
3. **비상 정지 버튼** 해제 상태 확인 (눌린 채로 있으면 시계 방향 해제)
4. 배터리 잔량 확인 (LED 주황색 Gradient = 저배터리)

### 충전이 안 됨

1. 충전 스테이션 LED 확인: 빨간색(대기) → 주황색(충전 중) 전환 여부
2. 충전 모드 스위치 → `O` (Auto) 확인
3. 충전 단자 접촉 상태 및 오염 확인
4. 충전 스테이션 주변 조도 확인 (300 Lux 이상)
5. 충전 스테이션 전원 코드 재연결 (약 20초 후)

### 도킹에 반복 실패

1. 충전 스테이션 앞 공간 확보 (1800 × 1600 mm)
2. 비주얼 마커 오염/가림 확인
3. 마커 높이 확인 (바닥 기준 **314 mm**, Baron 2.x 기준)
4. 주변 조도 확인 (직사광선, 반사광 차단)
5. 충전 스테이션 수평/벽 수직 상태 확인

### 로봇이 장애물 앞에서 멈추고 재개 안 됨 (Stuck)

1. **장애물 제거**
2. **Start 버튼** 눌러 재개
3. 그래도 안 되면: Joystick Manual 모드로 전환 후 수동 이동

### LED 빨간색 Cross Blink (시스템 오류)

1. 소프트웨어 Power 버튼으로 재부팅 시도
2. 재부팅 후에도 동일 오류: 하드웨어 전원 사이클 (하드웨어 스위치 OFF → ON)
3. 해결 안 되면: `sales@yujinrobot.com` 또는 담당 기술 지원팀 연락

### 조이스틱으로 로봇이 움직이지 않음

1. Joystick 스위치 → `Manual` 확인
2. 작동 스위치 → `X` 방향 확인 (`D` 방향이면 오작동 방지 모드)
3. 브레이크 스위치 → `ON` 확인
4. **LB(데드맨 스위치) 누른 상태**로 조작하는지 확인
