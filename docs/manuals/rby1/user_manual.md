# RB-Y1 사용자 매뉴얼

## 개요

RB-Y1은 모바일 매니퓰레이터 로봇이다. 이동 플랫폼과 양팔 조작 기능을 함께 사용하므로, 주행 안전과 작업 반경 안전을 동시에 확인해야 한다.

## 기동 절차

1. E-Stop이 해제되어 있는지 확인한다.  
2. TOM 전원을 ON 한다.  
3. RB-Y1 본체 전원을 켠다.  
4. 약 60초간 관절 초기화를 기다린다.  
5. `rby1_robot.py` 실행 상태를 확인한다.  
6. Rubberneck에서 영상과 제어 상태를 확인한다.

## 점검 체크리스트

- E-Stop 해제
- 관절 초기화 완료
- TOM 연결 상태
- Rubberneck 연결 상태
- 작업 반경 내 장애물 없음
- 그리퍼 상태 확인

## 수동 제어

1. Rubberneck에서 RB-Y1을 선택한다.  
2. 이동 제어와 양팔 제어 모드를 구분한다.  
3. 팔 제어 시 속도를 50% 이하로 제한한다.  
4. Space 키 또는 E-Stop으로 긴급 정지한다.

## 안전 주의

- 매니퓰레이터 작업 반경 안에 사람이 들어오면 즉시 정지한다.
- 관절 초기화 전 팔을 강제로 움직이지 않는다.
