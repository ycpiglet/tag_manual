# SOLAR 사용자 매뉴얼

## 개요

SOLAR는 로봇 Fleet 관리, 임무 스케줄링, 경로 계획을 위한 관제 시스템이다.

## 기동 절차

1. SOLAR 플랫폼에 접속한다.  
2. 로봇 Fleet 상태를 확인한다.  
3. 임무 스케줄을 설정한다.  
4. 경로와 작업 구역의 유효성을 확인한다.  
5. 자율 임무 실행 상태를 모니터링한다.

## 점검 체크리스트

- SOLAR 계정 권한 확인
- 로봇 연결 상태 확인
- 임무 경로 유효성 검사
- 비상 정지 버튼 접근성 확인
- 로그 저장 상태 확인
