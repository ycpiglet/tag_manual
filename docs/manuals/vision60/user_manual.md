# Vision60 사용자 매뉴얼

## 개요

Vision60은 4족 보행 로봇으로, 야외 및 발전소 환경 순찰·점검을 위한 플랫폼이다.

## 기동 절차

1. 배터리 상태를 확인한다.  
2. TOM 전원을 ON 한다.  
3. Vision60 본체 전원을 켠다.  
4. 보행 제어 초기화를 기다린다.  
5. Rubberneck 연결 상태를 확인한다.  
6. 현장 안전 구역을 확보한다.

## 점검 체크리스트

- 배터리 상태
- 보행 제어 상태
- TOM 연결 상태
- Rubberneck 연결 상태
- 카메라/센서 상태
- 야외 장애물 여부

## 비상 대응

- 보행 불안정 시 즉시 정지한다.
- 야외 통신 품질이 낮으면 원격 조작을 중지한다.
- 발전소 현장에서는 현장 관리자 지시에 따른다.
