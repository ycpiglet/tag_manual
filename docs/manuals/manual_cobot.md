# COBOT 사용자 매뉴얼

## 개요

COBOT은 협동로봇 기반 조작 작업을 위한 시스템이다. 사람과 가까운 거리에서 운용될 수 있으므로 안전 설정을 우선 확인해야 한다.

## 기동 절차

1. 전원과 안전 펜스 상태를 확인한다.  
2. TOM 전원을 ON 한다.  
3. COBOT 제어 프로세스를 실행한다.  
4. 원점 복귀 또는 초기 자세를 확인한다.  
5. 작업 프로그램을 선택한다.

## 점검 체크리스트

- 비상 정지 상태
- 속도 제한 설정
- 작업 반경 장애물 없음
- 그리퍼/툴 상태
- 제어 프로그램 상태

## 안전 주의

- 협동 운전 모드와 자동 운전 모드를 명확히 구분한다.
- 작업 반경 내 사람 접근 시 즉시 정지한다.
