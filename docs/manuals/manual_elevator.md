# 엘리베이터 MtoM IoT 매뉴얼

## 개요

엘리베이터 제어 모듈은 로봇의 층간 이동을 지원하기 위한 MtoM IoT 장치다.

## 기동 절차

1. TOM 전원을 ON 한다.  
2. `elevator_robot.py` 실행 상태를 확인한다.  
3. 층 호출 제어가 가능한지 확인한다.  
4. Rubberneck 연결 상태를 확인한다.  
5. 로봇 연동 모드를 ON 한다.

## 점검 체크리스트

- TOM 상태
- 제어 모듈 상태
- Rubberneck 연결 상태
- 층 호출 응답 상태
- 안전 인터록 상태

## 수동 제어

- Rubberneck에서 엘리베이터를 선택한다.
- 대상 층 호출 명령을 전송한다.
- 로봇 탑승/하차 상태를 확인한다.
