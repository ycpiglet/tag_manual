# 🤖 GoCart180 초보자 완벽 가이드: 러버넥(Rubberneck) 조종석 매뉴얼

안녕. 화면 앞 조종석에 앉은 걸 환영해. 
네 눈앞에 있는 거대한 로봇, GoCart180을 움직이는 방법은 생각보다 아주 단순해. 
어릴 적 하던 카트라이더나 자동차 게임을 떠올려봐. 철과 코드로 이루어진 녀석이지만, 결국 네 손끝에서 움직이는 하나의 RC카일 뿐이니까.

복잡한 이야기는 빼고, 당장 로봇을 내 맘대로 조종하기 위해 알아야 할 핵심만 직관적으로 짚어줄게.

---

## 🟢 1단계: 시동 걸기 (가장 먼저 해야 할 일)
로봇이 네 명령을 들을 준비를 하는 과정이야. 움직이기 전에 딱 두 번만 키보드를 눌러주면 돼.

* **`Q` : 로봇 깨우기 (로그인 / 로그아웃)**
  * 로봇에게 "내가 이제 너 조종할 거야"라고 인사하는 키야. 한 번 누르면 연결되고, 다 쓰고 나서 다시 누르면 연결이 끊어져.
* **`Z` : 수동 운전 시작 (조이스틱 On)**
  * "이제부터 내 맘대로 운전대 잡는다"라고 선언하는 키. **이걸 반드시 눌러야** 로봇이 네 방향키에 반응하기 시작해.
* **`C` : 수동 운전 끝 (조이스틱 Off)**
  * 내가 직접 운전하는 걸 그만두고 싶을 때 눌러.
* **`R` : 조종 모드 스위치 (On/Off 전환)**
  * `Z`와 `C`를 따로 누르기 귀찮다면, 그냥 `R`을 눌러서 껐다 켰다 할 수도 있어. (스마트폰 화면 자동회전 버튼처럼!)

---

## 🏎️ 2단계: 직접 운전하기 (W, A, S, D)
`Z`를 눌러 운전대를 잡았다면, 이제 키보드가 네 자동차의 엑셀과 핸들이 돼.

* **`W`** : **앞으로 가기 (엑셀 밟기)**
* **`S`** : **뒤로 가기 (후진 기어)**
* **`A`** : **왼쪽으로 돌기 (좌회전)**
* **`D`** : **오른쪽으로 돌기 (우회전)**

> 💡 **Tip:** 처음부터 키보드를 꾹 누르지 말고 살짝살짝 눌러봐. 덩치가 큰 만큼 아주 부드럽고 묵직하게 움직일 거야.

---

## 🎯 3단계: 로봇에게 심부름 시키기 (자동 임무 모드)
매번 땀 흘리며 직접 운전할 필요 없어. 버튼 하나면 알아서 목적지로 스스로 굴러가거든.

* **`Space`(스페이스바) 또는 `방향키 오른쪽(→)` : 집으로 돌아가기**
  * 놀 만큼 놀았거나 배터리를 충전해야 할 때 누르는 '귀환 버튼'이야. 로봇이 알아서 자기 집(홈)으로 찾아가.
* **`방향키 위(↑)` : 1층 순찰 돌기**
  * 정해진 1층 구역을 알아서 돌아다니며 순찰하는 임무를 줘. 
* **`E` : 로봇의 속마음 보기 (로그 켜기/끄기)**
  * 로봇이 지금 무슨 생각을 하는지, 어디가 아픈지 화면에 알 수 없는 영어(로그)로 띄워줘. 궁금할 때 한 번씩 껐다 켰다 해봐.

---

## 🚨 4단계: 위급 상황과 고급 기술 (조이패드 사용자 전용)
만약 키보드 말고 엑스박스 같은 게임용 조이패드가 연결되어 있다면, 더 고급스러운 조작도 가능해. (당장 몰라도 로봇을 쓰는 데는 지장 없어!)

* **잠깐 멈춤! (얼음과 땡)**
  * **`L2` (왼쪽 트리거)** : **얼음! (Soft Pause)** - 로봇이 부드럽게 제자리에 멈춰 서서 대기해.
  * **`R2` (오른쪽 트리거)** : **땡! (Soft Resume)** - 멈춰있던 로봇이 다시 하던 일을 계속해.
* **임무 강제 취소**
  * **`Back` 버튼** : 하던 일을 당장 멈추고 취소해버려. 로봇이 위험한 곳으로 가려 할 때 누르는 비상브레이크야.
* **전문가용 배달 임무**
  * **`Start` 버튼** : 미리 짜여진 무한 반복 임무를 시작해.
  * **`왼쪽 스틱 꾹 누르기`** : 1층과 5층을 오가는 배달 미션 시작.
  * **`오른쪽 스틱 꾹 누르기`** : 1층과 22층을 오가는 배달 미션 시작.

---

### 📝 3초 요약 (이것만 기억해!)
처음 만났다면 딱 순서대로 세 개만 눌러봐.
1. **`Q`** 로 인사하고,
2. **`Z`** 로 운전대를 쥔 다음,
3. **`W, A, S, D`** 로 맘껏 달려보는 거야. 

복잡한 건 나중에 생각하고, 일단 부딪혀봐. 실전이 최고의 스승이니까.
