# SPOT 사용자 매뉴얼

## 개요

SPOT은 4족 보행 기반 순찰·점검 로봇이다. 인천 청라 로봇랜드와 영월 현장 등에서 원격 관제와 현장 순찰 용도로 운용한다.

## 기동 절차

1. 배터리 잔량이 30% 이상인지 확인한다.  
2. TOM 전원을 ON 한다.  
3. SPOT 본체 전원을 켠다.  
4. 기립 대기 상태에서 약 30초간 초기화를 기다린다.  
5. `spot_robot.py` 실행 상태를 확인한다.  
6. Rubberneck에서 영상과 제어 연결을 확인한다.

## 점검 체크리스트

- 배터리 30% 이상
- TOM 전원 상태
- E-Stop 해제 상태
- Rubberneck 연결 상태
- 카메라 영상 수신 상태
- SOLAR 연동 상태

## 수동 제어

1. Rubberneck에서 SPOT을 선택한다.  
2. WASD 또는 조이스틱으로 이동한다.  
3. Space 키로 정지한다.  
4. 임무 종료 후 앉기 명령을 수행한다.  
5. 안전 확인 후 전원을 차단한다.

## 비상 대응

- 이상 보행 발생 시 즉시 정지한다.
- 통신 지연이 크면 원격 조작을 중단한다.
- E-Stop 동작 후 재기동 전 주변 장애물을 확인한다.
