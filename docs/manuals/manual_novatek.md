# Novatek 사용자 매뉴얼

## 개요

Novatek은 AMR 계열 로봇으로, 맵 기반 주행과 현장 임무 테스트에 사용한다.

## 기동 절차

1. 배터리 상태를 확인한다.  
2. TOM 전원을 ON 한다.  
3. Novatek 제어 프로세스를 실행한다.  
4. 맵 로딩 상태를 확인한다.  
5. Rubberneck 연결을 확인한다.

## 점검 체크리스트

- 배터리 상태
- 맵 파일 상태
- TOM 연결 상태
- Rubberneck 연결 상태
- 주행 경로 장애물 여부

## 운영 주의

- 맵 업데이트 후에는 저속으로 시험 주행한다.
- 경로 변경 시 기존 waypoint와 충돌하지 않는지 확인한다.
