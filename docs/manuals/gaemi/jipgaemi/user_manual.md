# 집개미 사용자 매뉴얼

## 개요

집개미는 실내 환경에서 운용되는 AMR 플랫폼이다. 일개미와 유사한 운용 절차를 사용하되, 기종별 파라미터와 경로 설정을 별도로 관리한다.

## 기동 절차

1. 충전 상태를 확인한다.  
2. TOM 전원을 ON 한다.  
3. 로봇 본체 전원을 ON 한다.  
4. `jipgaemi_robot.py` 실행 상태를 확인한다.  
5. Rubberneck 영상과 제어 상태를 확인한다.

## 점검 체크리스트

- 배터리 상태
- TOM 상태
- Rubberneck 상태
- SOLAR 상태
- 주행 경로 장애물 여부

## 수동 제어

- Rubberneck에서 집개미를 선택한다.
- 키보드 또는 조이스틱으로 수동 조작한다.
- 긴급 상황에서는 즉시 정지한다.
