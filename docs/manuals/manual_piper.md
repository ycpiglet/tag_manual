# PiPER 사용자 매뉴얼

## 개요

PiPER는 매니퓰레이터 기반 조작 실험과 원격 제어 검증에 사용하는 로봇 암이다.

## 기동 절차

1. 전원과 통신 케이블을 확인한다.  
2. TOM 전원을 ON 한다.  
3. PiPER 제어 프로세스를 실행한다.  
4. 관절 초기화 상태를 확인한다.  
5. Rubberneck 또는 로컬 제어 인터페이스에서 제어 권한을 확인한다.

## 점검 체크리스트

- 전원 상태
- 관절 초기화 상태
- 제어 프로세스 상태
- 작업 반경 장애물 없음
- 비상 정지 가능 여부

## 안전 주의

- 팔 주변 작업 반경을 확보한다.
- 고속 동작은 테스트 환경에서만 수행한다.
- 그리퍼 작업 시 손 끼임 위험을 확인한다.
