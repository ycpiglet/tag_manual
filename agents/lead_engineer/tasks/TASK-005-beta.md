담당: Beta Tester

## 목표

서버가 기동된 후, 기술 지식 없는 일반 사용자처럼 앱을 자유롭게 탐색하여
예상치 못한 에러와 이상 동작을 발견한다.

## 시작 조건

TASK-001(Backend) 완료 후 서버가 기동된 상태에서 시작.

접근 URL: `http://localhost:8080/public/index.html`

## 탐색 방식

매뉴얼을 읽지 않는다. 보이는 것을 누른다.
계획 없이 이것저것 해본다.

## 최소 탐색 시나리오

- [ ] 로그인 화면에서 아무것도 입력 안 하고 로그인 버튼 클릭
- [ ] 이메일 형식이 아닌 값을 ID 필드에 입력
- [ ] 매우 긴 비밀번호 입력 (100자 이상)
- [ ] 로그인 후 모든 탭을 빠르게 클릭
- [ ] 같은 버튼을 연속으로 여러 번 클릭
- [ ] 브라우저 뒤로 가기 버튼 사용
- [ ] URL을 직접 입력해서 다른 탭에 접근 시도
- [ ] 화면 크기를 매우 작게 줄임 (모바일 너비)
- [ ] 화면 크기를 매우 크게 키움
- [ ] 로봇 카드를 하나씩 전부 클릭
- [ ] 다크모드 켜고 끄기를 빠르게 반복
- [ ] PDF 내보내기 여러 번 연속 클릭

## 발견 기록

이상한 점, 에러, 예상과 다른 동작은 즉시:
`agents/beta_tester/test_cases/BTC-{번호}.md` 에 기록
`agents/beta_tester/test_cases/INDEX.md` 업데이트
