# BTC-001

발견일: 2026-05-13
제목: 화면을 좁게 줄이면 앱이 잘려서 옆으로 스크롤해야 함
심각도 (사용자 관점): 불편하다
재현 가능: 항상

---

## 행동

브라우저 창 너비를 모바일처럼 좁게 줄여봤어요 (약 400~700px 정도).

## 결과

앱이 1040px 이하로 줄어들지 않고, 화면 오른쪽이 잘린 채로 가로 스크롤바가 생겼어요.
사이드바, 버튼들이 화면 밖으로 밀려났어요.
휴대폰이나 태블릿으로 보면 쓰기가 정말 어려울 것 같아요.

## 기대

화면 크기에 맞게 레이아웃이 줄어들거나, 최소한 주요 기능은 사용할 수 있어야 할 것 같아요.

## 기술 메모 (코드 확인 결과)

`body { min-width: 1040px }` 가 CSS에 고정되어 있음.
반응형 breakpoint(@media max-width)가 1100px 이하에서 일부 그리드만 조정하고,
body 자체의 최소 너비를 해제하지 않음.
이 앱은 현재 데스크탑 전용으로 설계된 상태.
