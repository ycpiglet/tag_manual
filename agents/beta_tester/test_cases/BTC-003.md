# BTC-003

발견일: 2026-05-13
제목: 새로고침(F5)해도 로그인이 유지됨 — 예상과 달랐음
심각도 (사용자 관점): 이상하다 (좋은 의미이기도 함)
재현 가능: 항상

---

## 행동

로그인 후 브라우저에서 F5를 눌러 새로고침했어요.

## 결과

로그인 화면이 다시 안 뜨고 바로 앱 화면이 보였어요.
브라우저를 완전히 닫고 다시 열어도 마찬가지였어요.

## 기대

새로고침하면 다시 로그인해야 할 줄 알았어요.
(다른 앱들은 보통 새로고침하면 로그인이 풀리는 경우가 있어서요)

## 기술 메모 (코드 확인 결과)

`localStorage`에 `star_team_auth_user_id` 키로 로그인 상태가 저장됨.
페이지 로드 시 `restoreSession()`이 자동 실행되어 세션 복원.
이 동작은 의도된 설계이나, 공용 PC에서 사용 시
"로그아웃 없이 자리 비움 → 다음 사용자가 이전 계정 상태로 접속" 시나리오 가능.
