## 행동

로그인 화면을 처음 봤을 때, 화면 아래쪽에 계정 정보가 안내로 적혀 있었어요.
그리고 브라우저에서 "페이지 소스 보기"를 눌러봤어요.

## 결과

로그인 화면 하단에 `admin / admin`, `itp_user / 1234`, `viewer / view` 같은
아이디와 비밀번호가 화면에 그대로 표시되고 있어요.
페이지 소스에서도 모든 계정 정보 (비밀번호 포함)가 텍스트로 보여요.

## 기대

비밀번호는 화면에 보이면 안 될 것 같아요.
다른 사람이 이 페이지를 열면 모든 계정을 알 수 있을 것 같아요.

## 기술 메모 (코드 확인 결과)

로그인 힌트 박스(`.login-hint`)가 HTML에 항상 렌더링됨.
프로토타입/내부 배포용으로는 허용 가능하나,
외부 공개 배포 전 반드시 제거 또는 서버사이드 인증으로 대체 필요.
