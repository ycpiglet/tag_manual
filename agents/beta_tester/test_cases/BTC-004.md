# BTC-004

발견일: 2026-05-13
제목: 내 위치 버튼이 외부 IP 서버에서 접속하면 동작 안 할 수 있음
심각도 (사용자 관점): 불편하다
재현 가능: 외부 IP + HTTP 환경에서만

---

## 행동

앱에서 위치 관련 버튼(GPS/내 위치)을 눌러봤어요.
localhost에서는 동작했는데, 외부 서버 주소(예: 192.168.x.x)로 접속 시
동작하지 않을 것 같아서 확인해봤어요.

## 결과

현재는 `localhost:8080`에서 서빙 중이라 GPS가 동작함.
하지만 같은 파일을 `http://192.168.x.x:8080` 같은 내부 네트워크 IP로
접속하면 브라우저가 위치 정보 접근을 차단함 (HTTP + non-localhost).

## 기대

어디서 접속해도 위치 기능이 동작하거나, 안 되면 명확한 안내가 나왔으면 해요.

## 기술 메모 (코드 확인 결과)

코드에 `isSecureContext || localhost || 127.0.0.1` 체크 로직 존재 (line ~3267).
외부 IP + HTTP 환경에서는 `gpsFallbackLabel()`로 폴백 처리됨 — 기능은 비활성화되지만
앱 자체가 깨지지는 않음. 외부 배포 시 HTTPS 적용 또는 GPS 기능 비노출 권장.
