🤖 STAR TEAM 이기종 로봇 운영 매뉴얼

KETI AX연구본부 지능로보틱스연구센터 · 인천 청라 · 서울 강남 · 강원 영월

TOM 모듈 기반 이기종 로봇 원격 관제 시스템입니다. Rubberneck으로 실시간 수동 제어, SOLAR로 자율 임무·Fleet 관리를 제공합니다.

📡
TOM
Tele-Operation Module · 로봇 부착형 원격제어 모듈
로봇에 부착하는 원격제어 하드웨어 모듈입니다. 5G/LTE 통신과 전원 ON 기반 자동 동작을 통해 Rubberneck 원격 관제의 현장 단말 역할을 수행합니다.
담당:
🔌
AND
Network Daemon · Rubberneck 자동 연결 데몬
TOM과 Rubberneck 서버 사이의 네트워크 연결을 담당합니다. WebSocket 기반 연결, heartbeat, 자동 등록 등 원격 관제 통신의 연결 계층을 관리합니다.
담당:
🎮
GERRI
Control Framework · 로봇 명령 처리 프레임워크
Rubberneck에서 전달된 원격 조작 명령을 로봇별 SDK·드라이버 호출로 변환하는 제어 프레임워크입니다. 로봇별 제어 로직과 안전 절차를 연결합니다.
담당:
🌐
Rubberneck
실시간 원격 관제 · www.rubberneck.kr
웹 브라우저에서 카메라 영상 수신 및 WASD 수동 제어. 계정 권한별 로봇 접근 제한.
담당:
☀️
SOLAR
Fleet 관리 · 임무 스케줄링 · 경로 계획
자율 임무 운영 플랫폼. 다수 로봇의 경로 계획, 상태 모니터링 통합 관리. 연동 준비 중.
담당:
🏛
KETI · STAR TEAM
한국전자기술연구원 지능로보틱스연구센터
이기종 로봇 원격 관제 아키텍처 연구·개발. 3개 사이트 운영 중.
총괄:
🗺️ 사이트별 로봇 배치

📡 실시간 영상 수신
TOM 부착 로봇의 카메라 영상을 지연 없이 웹 브라우저로 수신합니다. 다중 로봇 동시 모니터링 지원.
🕹 원격 수동 제어
WASD / 조이스틱으로 브라우저에서 직접 로봇 이동 제어. 계정 권한에 따라 접근 가능한 로봇이 구분됩니다.
🔗 TOM 연동
AND 데몬이 WebSocket으로 Rubberneck 서버에 연결됩니다. TOM 전원 ON 즉시 자동 등록.
👤 계정 권한 관리
관리자 계정에서 사용자별 로봇 접근 권한을 설정합니다. STAR TEAM 연락처로 계정 신청.
