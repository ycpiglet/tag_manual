📡
TOM
Tele-Operation Module · 로봇 부착형 원격제어 모듈
로봇에 부착하는 원격제어 하드웨어 모듈입니다. 5G/LTE 통신과 전원 ON 기반 자동 동작을 통해 Rubberneck 원격 관제의 현장 단말 역할을 수행합니다.

🔌
AND
Network Daemon · Rubberneck 자동 연결 데몬
TOM과 Rubberneck 서버 사이의 네트워크 연결을 담당합니다. WebSocket 기반 연결, heartbeat, 자동 등록 등 원격 관제 통신의 연결 계층을 관리합니다.
담당:
🎮
GERRI
Control Framework · 로봇 명령 처리 프레임워크
Rubberneck에서 전달된 원격 조작 명령을 로봇별 SDK·드라이버 호출로 변환하는 제어 프레임워크입니다. 로봇별 제어 로직과 안전 절차를 연결합니다.
담당:
Rubberneck
실시간 원격 관제 · www.rubberneck.kr

SOLAR
Fleet 관리 · 임무 스케줄링 · 경로 계획
자율 임무 운영 플랫폼. 다수 로봇의 경로 계획, 상태 모니터링 통합 관리. 연동 준비 중.
