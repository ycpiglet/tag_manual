## 0. 1분 운영 대시보드

> [!IMPORTANT]
> **현장 원칙은 단순하다.** 사람이 위험하면 먼저 멈춘다. 로봇이 이상하면 LED를 본다. 움직이지 않으면 `Brake`, `Joystick`, `E-Stop`, `Battery` 순서로 확인한다.

| 상황 | 먼저 볼 것 | 정상 기준 | 바로 조치 |
|---|---|---|---|
| 🟢 정상 주행 | LED 밴드 | 하늘색 점등 또는 깜빡임 | 주변 안전거리 유지 |
| ⚪ 부팅 중 | LED 밴드 | 흰색 0.5s 깜빡임 | 2~3분 대기 후 [[Start]] |
| 🔴 급정지 | E-Stop, LED | 빨간색 점등 | 주변 확인 → E-Stop 해제 → [[Start]] |
| 🟠 저배터리 | LED, 충전 상태 | 주황색 Gradient | 즉시 충전 위치로 이동 |
| 🕹️ 수동 조작 | Joystick SW, [[LB]] | Manual + [[LB]] 누름 | 조작 후 Auto 복귀 |
| 🔌 충전 실패 | 충전 스테이션 LED | 주황색=충전 중 | 단자/모드/공간/조도 확인 |

### 빠른 판단 흐름

```text
문제 발생
  ↓
사람·장애물 위험? ── 예 → E-Stop 누름 → 주변 확인 → 해제 → Start
  ↓ 아니오
LED 색상 확인 → 빨강/주황/노랑이면 오류·배터리·위치 문제 우선 확인
  ↓
로봇이 안 움직임 → Brake ON → Joystick Auto → E-Stop 해제 → 배터리 확인
  ↓
그래도 안 됨 → 재부팅 또는 기술 지원 요청
```

## 2. 전원 켜기/끄기

### 2.1 켜기 순서

> [!NOTE]
> 총 소요 시간은 보통 **2~3분**이다. 부팅이 끝나도 [[Start]]를 누르기 전까지 로봇은 명령을 받지 않는다.

| 단계 | 동작 | 확인 포인트 |
|---:|---|---|
| 1 | 후면 배터리 덮개 열기 | 열쇠 사용 |
| 2 | 트리거 스위치 → `I` | 기본값. 보통 건드릴 필요 없음 |
| 3 | 하드웨어 전원 스위치 → `ON` | 실제 전원 공급 시작 |
| 4 | 배터리 덮개 닫기 | 케이블 끼임 확인 |
| 5 | 소프트웨어 [[Power]] 3초 이상 누르기 | 부팅 시작 |
| 6 | [[Start]] LED 녹색 깜빡임 확인 | 명령 수신 준비 상태 |
| 7 | [[Start]] 누르기 | 로봇 작동 시작 |

```text
배터리 덮개 → Trigger I → HW Power ON → SW Power 3초 → Start LED 녹색 점멸 → Start
```

### 2.2 끄기 순서

| 상황 | 순서 | 언제 사용하나 |
|---|---|---|
| 일반 종료 | [[Power]] 3초 이상 → LED 완전 소등까지 약 1분 대기 | 당일 운용 종료 |
| 장기 미사용/점검 | 일반 종료 → 하드웨어 스위치 `OFF` → 트리거 `O` | 장기 보관, 물리 점검, 운송 전 |

> [!WARNING]
> 장기 보관 시 트리거 스위치까지 끄지 않으면 배터리가 자가 방전될 수 있다.

## 3. 비상 정지

### 한눈에 보기

| 동작 | 결과 | 다음 행동 |
|---|---|---|
| 좌측/우측 E-Stop 누름 | 즉시 모터 전원 차단 | 주변 안전 확인 |
| E-Stop 시계 방향 회전 | 버튼 잠금 해제 | [[Start]] LED 확인 |
| [[Start]] 누름 | 정상 복귀 | 저속으로 동작 확인 |

> [!IMPORTANT]
> E-Stop 해제는 “로봇을 다시 움직여도 안전하다”는 판단 이후에만 한다. 버튼 해제가 곧 운전 허가가 아니다.

### 복귀 절차

1. 비상 정지 버튼을 **시계 방향으로 돌려** 잠금 해제
2. [[Start]] 버튼 LED가 **녹색으로 깜빡이는지** 확인
3. 주변 사람, 장애물, 단차를 확인
4. [[Start]] 누르기
5. 저속 동작으로 정상 복귀 확인

## 4. 브레이크 스위치

| 스위치 상태 | 의미 | 로봇 동작 | 현장 해석 |
|---|---|---|---|
| `ON` | 브레이크 정상 작동 | 자율/수동 주행 가능 | **운용 기본값** |
| `OFF` | 모터 전원 차단 | 주행 불가, 손으로 밀기만 가능 | 수동 이동·긴급 이동용 |

### 수동으로 밀어서 이동할 때

1. 로봇이 완전히 정지했는지 확인
2. Brake SW → `OFF`
3. 손으로 천천히 밀기
4. 이동 완료 후 반드시 Brake SW → `ON`
5. [[Start]] 또는 관제 상태로 정상 복귀 확인

> [!CAUTION]
> 급하게 밀면 감속 충격이 발생할 수 있다. 특히 적재물이 있을 때는 로봇보다 화물이 먼저 움직이는 상황이 생길 수 있다.

## 5. 조이스틱 조작 Logitech

### 5.1 조작 전 체크리스트

- [ ] Joystick SW가 `Manual` 모드인지 확인
- [ ] [[Start]] 버튼으로 로봇이 활성화되었는지 확인
- [ ] 조이스틱 전면 작동 스위치가 `X` 방향인지 확인
- [ ] 주변 사람, 케이블, 장애물을 확인
- [ ] 수동 주행 후 Joystick 열쇠를 제거할 준비가 되었는지 확인

> [!WARNING]
> 조이스틱 전면 스위치가 `D` 방향이면 오작동 방지 상태일 수 있다. 움직이지 않을 때는 `X` 방향부터 확인한다.

### 5.2 버튼 역할 지도

| 버튼/입력 | 역할 | 기억법 |
|---|---|---|
| [[LB]] | 데드맨 스위치. 항상 누른 상태로 사용 | “누르고 있어야 산다” |
| 모드 버튼 | D-pad/아날로그 스틱 전환 | 녹색 LED 상태 확인 |
| D-pad | 키패드 방향 조작 | 짧고 정확한 이동 |
| 좌 스틱 | 아날로그 조작 | 부드러운 이동 |
| [[RT]] | 터보 모드 | 넓은 공간에서만 사용 |
| Logitech 버튼 | SLAM/Localization 모드 전환 | 지도 작성/위치 추정 전환 |
| 우 스틱 | 기능 없음 | 사용하지 않음 |

### 5.3 조작 모드 선택

| 원하는 조작 | 모드 버튼 LED | 입력 장치 | 조작 방법 |
|---|---|---|---|
| D-pad 조작 | 녹색 LED 켜짐 | D-pad | [[LB]] 누른 상태에서 방향 입력 |
| 아날로그 조작 | 녹색 LED 꺼짐 | 좌 스틱 | [[LB]] 누른 상태에서 스틱 조작 |

### 5.4 매핑 SLAM 권장 절차

```text
Manual 전환 → Start → SLAM 모드 → 외곽 먼저 주행 → 내부 구역 주행 → 원점 통과 → Localization 전환
```

1. Joystick SW → `Manual`
2. [[Start]]로 로봇 활성화
3. Logitech 버튼으로 **SLAM 모드** 선택
4. 로봇 뒤에 서서 따라가며 공간 전체를 주행
5. 외곽 경로 먼저 주행한 뒤 내부 구역 주행
6. 내부 구역 경로가 시작 지점, 즉 원점을 통과하도록 설계
7. 매핑 완료 후 Logitech 버튼으로 **Localization 모드** 전환

> [!TIP]
> 매핑은 청소기처럼 “빠짐없이 훑는 작업”에 가깝다. 먼저 외곽선을 그리고, 그다음 내부를 색칠하듯 채우면 지도 품질이 좋아진다.

## 6. LED 상태 표시

### 6.1 LED 판단 우선순위

| 우선순위 | 색상 | 의미 | 운영자 행동 |
|---:|---|---|---|
| 1 | 🔴 빨간색 | 정지/오류/기능 안전 문제 | 즉시 원인 확인 |
| 2 | 🟠 주황색 | 저배터리 | 충전 우선 |
| 3 | 🟡 노란색 | 위치 추정 문제 | 로컬라이제이션 재확인 |
| 4 | ⚪ 흰색 | 부팅/대기/일시 정지 | 상태 변화 대기 또는 입력 |
| 5 | 🔵 하늘색 | 주행/수동/SLAM | 정상 운용 |
| 6 | 🟢 녹색 | 충전/완충 | 충전 상태 확인 |

### 6.2 자주 보는 상태

| LED 표시 | 상황 | 조치 |
|---|---|---|
| 🔵 하늘색 Stay On | 자율 주행 또는 수동 주행 | 주변 안전거리 유지 |
| 🔵 하늘색 0.5s Blink | 전후진/회전 중 | 이동 방향 확인 |
| 🔵 하늘색 0.3s Blink | 고속 주행 중 | 사람 접근 주의 |
| ⚪ 흰색 0.5s Blink | 부팅 중 | 2~3분 대기 |
| ⚪ 흰색 Stay On | 일시 정지 | 재개 조건 확인 |
| 🟢 녹색 Gradient | 충전 중 | 충전 단자 접촉 유지 |
| 🟢 녹색 Stay On | 완충 완료 | 운용 가능 |
| 🟠 주황색 Gradient | 저배터리 | 즉시 충전 |
| 🔴 빨간색 Stay On | 비상 정지 활성화 | E-Stop 해제 후 [[Start]] |
| 🔴 빨간색 Cross Blink | 긴급 레벨 오류 | 재부팅 또는 지원 요청 |
| 🔴 빨간색 0.5s Cross Blink | FuSa 오류 | 라이다/인코더/GPIO/Cliff 관련 확인 |
| 🔴 빨간색 Burst | Stuck, 장애물에 막힘 | 장애물 제거 후 [[Start]] |
| 🟡 노란색 Cross Burst | 로컬라이제이션 미완료 | 충전 스테이션 복귀 후 재시도 |

| 빨간색 0.5s Blink | 도킹 실패 또는 BMS 오류 | 충전 위치 재확인 |
| 주황색 Gradient | 저배터리 | 즉시 충전 |
| 노란색 Cross Burst | 로컬라이제이션 미완료 | 충전 스테이션 복귀 후 재시도 |
:::

## 7. 충전

### 7.1 충전 방식 선택표

| 상황 | 권장 방식 | 판단 기준 |
|---|---|---|
| 정상 운영 중 | 자동 충전 | FMS 또는 Rubberneck에서 도킹 가능 |
| 자동 도킹 실패 | 수동 충전 | 충전 스테이션은 사용 가능하지만 도킹이 어려움 |
| 충전 스테이션 사용 불가 | 배터리 수동 충전기 | 스테이션 장애, 완전 방전, 현장 복구 필요 |

### 7.2 자동 충전

```text
FMS/Rubberneck 도킹 명령 → 로봇 접근 → 충전 단자 접촉 → 스테이션 LED 주황색 → 완충 시 녹색
```

자동 충전 전 확인할 것:

- [ ] 충전 모드 스위치 → `O`, Auto Mode
- [ ] 충전 스테이션 전원 케이블 연결
- [ ] 충전 단자와 로봇 단자 높이 일치
- [ ] 충전 스테이션 주변 **1800 × 1600 mm** 공간 확보
- [ ] 조도 **300 Lux 이상** 확보
- [ ] 비주얼 마커 오염 또는 가림 없음

### 7.3 수동 충전

> [!WARNING]
> 수동 충전은 “비상 시 우회 절차”다. 완료 후 충전 모드 스위치를 반드시 `O`, Auto Mode로 복구한다.

:::details 수동 충전 절차 펼치기
1. 로봇과 충전 스테이션 전원 모두 끄기  
   - 로봇: 하드웨어 스위치 → `OFF`  
   - 충전 스테이션: 전원 케이블 분리
2. 충전 스테이션 충전 모드 스위치 → `I`, Manual Mode
3. 케이블 브라켓에서 수동 충전 케이블 분리
4. 수동 충전 케이블을 로봇 충전 커넥터에 연결  
   - 빨간색 점 표시 일치 확인
5. 로봇과 충전 스테이션 전원 켜기
6. 충전 스테이션 LED가 주황색으로 변경되는지 확인
7. 완전 충전 후 전원 끄기
8. 케이블 분리
9. 충전 모드 스위치 → `O`, Auto Mode로 복구
:::

### 7.4 배터리 수동 충전기 사용

> [!IMPORTANT]
> GoCart180은 **29 V 충전기**를 사용한다. 충전기 전압이 다르면 사용하지 않는다.

:::details 배터리 수동 충전기 절차 펼치기
1. 배터리 덮개 열기
2. 하드웨어 스위치 → `OFF`
3. 트리거 스위치 → `O`
4. 배터리 패널에서 로봇 전원 케이블 커넥터 분리
5. 배터리 수동 충전기 케이블을 배터리 소켓에 연결
6. 충전기 전원 플러그를 220V 콘센트에 연결
7. 전원 어댑터 LED가 녹색 → 빨간색으로 바뀌면 충전 시작  
   - 녹색이 계속 유지되면 배터리 불량 가능성 있음
8. 완전 방전 상태면 최소 5시간 이상 충전
9. 충전 완료 후 충전기 분리
:::

### 7.5 충전 스테이션 LED 의미

| LED 색상 | 상태 | 해석 |
|---|---|---|
| 🔴 빨간색 점등 | 충전 대기 | 로봇 없음 또는 접촉 전 |
| 🔴 빨간색 점멸 | 비정상 상태 | 스테이션/접촉 상태 확인 |
| 🟠 주황색 점등 | 충전 중 | 정상 충전 |
| 🟢 녹색 점등 | 완충 완료 | 운용 가능 |
| 🟢 녹색 점멸 | 로봇 부팅 중 | 부팅 완료까지 대기 |

## 8. 배터리 완전 방전 시 복구

### 증상

- 배터리 잔량 0%
- 소프트웨어 [[Power]] 버튼 무반응
- 자동 도킹 불가

### 복구 흐름

```text
HW OFF → Trigger I/O/I 리셋 → HW ON → 수동 충전 시작 → 30분 이상 충전 → 24V 이상 확인 → 자동 충전 복귀
```

### 절차

1. 배터리 덮개 열기
2. 하드웨어 스위치 → `OFF`
3. 트리거 스위치 리셋  
   `I` → `O` → `I`
4. 하드웨어 스위치 → `ON`
5. 배터리 덮개 닫기
6. [7.3 수동 충전](#73-수동-충전) 방식으로 초기 충전 시작
7. 최소 **30분 충전**
8. 배터리 전압 **24 V 이상** 확인
9. 자동 충전 방식으로 전환

> [!CAUTION]
> 완전 방전은 배터리 수명에 좋지 않다. 주황색 Gradient가 보이면 “나중에”가 아니라 “지금” 충전한다.

## 11. 트러블슈팅

### 11.1 증상별 빠른 조치표

| 증상 | 1차 확인 | 2차 확인 | 해결 방향 |
|---|---|---|---|
| Start 후에도 안 움직임 | Brake `ON` | Joystick `Auto`, E-Stop 해제, 배터리 | 상태 복구 후 [[Start]] |
| 충전이 안 됨 | 스테이션 LED | Auto Mode `O`, 단자 오염, 조도 | 접촉/환경 복구 |
| 도킹 반복 실패 | 앞 공간 1800 × 1600 mm | 마커 오염, 높이 314 mm, 조도 | 스테이션 환경 정리 |
| 장애물 앞에서 멈춤 | 장애물 제거 | [[Start]] 재개 | 필요 시 Manual 이동 |
| 빨간색 Cross Blink | 소프트웨어 재부팅 | 하드웨어 전원 사이클 | 지원팀 연락 |
| 조이스틱 불응 | Manual 모드 | `X` 방향, Brake `ON`, [[LB]] | 조작 조건 복구 |

:::details 로봇이 Start 버튼 후에도 움직이지 않음
1. Brake SW → `ON` 확인
2. Joystick SW → `Auto` 확인
3. E-Stop 버튼 해제 상태 확인  
   - 눌린 채로 있으면 시계 방향으로 해제
4. 배터리 잔량 확인  
   - LED 주황색 Gradient = 저배터리
5. 조건 복구 후 [[Start]] 누르기
:::

:::details 충전이 안 됨
1. 충전 스테이션 LED 확인  
   - 빨간색 대기 → 주황색 충전 중으로 바뀌는지 확인
2. 충전 모드 스위치 → `O`, Auto 확인
3. 충전 단자 접촉 상태 및 오염 확인
4. 충전 스테이션 주변 조도 확인  
   - 300 Lux 이상 권장
5. 충전 스테이션 전원 코드 재연결  
   - 약 20초 후 재확인
:::

:::details 도킹에 반복 실패
1. 충전 스테이션 앞 공간 확보  
   - 1800 × 1600 mm
2. 비주얼 마커 오염/가림 확인
3. 마커 높이 확인  
   - 바닥 기준 314 mm, Baron 2.x 기준
4. 주변 조도 확인  
   - 직사광선, 반사광 차단
5. 충전 스테이션 수평 및 벽 수직 상태 확인
:::

:::details 로봇이 장애물 앞에서 멈추고 재개 안 됨 Stuck
1. 장애물 제거
2. [[Start]] 눌러 재개
3. 그래도 안 되면 Joystick Manual 모드로 전환
4. 저속으로 안전 위치까지 수동 이동
5. Joystick SW를 `Auto`로 복구
:::

:::details LED 빨간색 Cross Blink 시스템 오류
1. 소프트웨어 [[Power]] 버튼으로 재부팅 시도
2. 재부팅 후에도 동일하면 하드웨어 전원 사이클 수행  
   - 하드웨어 스위치 `OFF` → `ON`
3. 해결 안 되면 `sales@yujinrobot.com` 또는 담당 기술 지원팀 연락
:::

:::details 조이스틱으로 로봇이 움직이지 않음
1. Joystick SW → `Manual` 확인
2. 조이스틱 작동 스위치 → `X` 방향 확인  
   - `D` 방향이면 오작동 방지 모드 가능
3. Brake SW → `ON` 확인
4. [[LB]] 데드맨 스위치를 누른 상태로 조작하는지 확인
5. 조작 후 Joystick SW → `Auto` 복구
:::

## 12. 현장 용어 미니 사전

| 용어 | 풀네임/의미 | 현장 설명 |
|---|---|---|
| E-Stop | Emergency Stop | 비상 정지. 누르면 모터 전원이 즉시 차단됨 |
| SLAM | Simultaneous Localization and Mapping | 지도를 만들면서 동시에 자기 위치를 추정하는 기술 |
| Localization | 위치 추정 | 이미 있는 지도 안에서 로봇의 현재 위치를 찾는 과정 |
| FuSa | Functional Safety | 기능 안전. 센서/제어계가 위험 상황에서 안전하게 동작하도록 하는 체계 |
| BMS | Battery Management System | 배터리 상태, 전압, 충전 상태를 관리하는 시스템 |
| Docking | 도킹 | 충전 스테이션에 로봇이 자동으로 붙는 동작 |
| Gradient | 그라디언트 | LED 밝기나 색이 부드럽게 변하는 패턴 |
| Stay On | 지속 점등 | 깜빡이지 않고 계속 켜져 있는 상태 |
| Blink | 점멸 | 일정 주기로 깜빡이는 상태 |

## 운영자 마지막 확인

- 위험하면 **E-Stop**.
- 안 움직이면 **Brake → Joystick → E-Stop → Battery**.
- 충전이 안 되면 **Mode → 단자 → 공간 → 조도**.
- 빨간색 LED는 “일단 멈추고 원인 확인”.
- 수동 조작 후에는 반드시 **Auto 복귀**.
