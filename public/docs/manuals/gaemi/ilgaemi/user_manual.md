# 일개미 사용자 매뉴얼

## 개요

일개미는 실내 AMR 플랫폼으로, 물류·순찰·연구 실험용 이동체로 운용한다.

## 기동 절차

1. 충전 상태를 확인한다.  
2. 충전 도킹에서 분리한다.  
3. TOM 전원을 ON 한다.  
4. 로봇 본체 전원을 ON 한다.  
5. `ilgaemi_robot.py` 실행 상태를 확인한다.  
6. Rubberneck 연결을 확인한다.

## 점검 체크리스트

- 배터리 80% 권장
- 경로상 장애물 없음
- TOM 연결 상태
- Rubberneck 연결 상태
- SOLAR 연결 상태

## 수동 제어

- Rubberneck 수동 모드로 전환한다.
- 조이스틱 또는 키보드로 이동한다.
- 이상 상황 시 긴급 정지한다.
